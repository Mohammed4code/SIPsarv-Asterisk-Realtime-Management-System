# SIPsarv

SIPsarv is a web-based Asterisk management platform built with Go and MySQL. It provides a simple interface for managing SIP extensions, users, and Asterisk configurations through PJSIP Realtime and AMI integration.

## Features

- SIP extension management
- User and contact management
- PJSIP Realtime support
- Asterisk AMI integration
- Real-time service monitoring
- REST API backend
- Web-based administration interface
